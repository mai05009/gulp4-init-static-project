# gulp4-init-static-project
使用gulp4初始化静态项目
目录结构：
--项目名
  --dist    //编译后目录
  --lib     //插件、框架像JQ，bootstrap
  +--src    //静态文件目录
    --css    //scss
    --img     
    --js
  --test    //本地运行文件，用于实时预览
  --view   //存放静态.html
  
项目使用gulp4，
完成热加载，代理，gulp-file-include，
cnpm i初始化后，运行npm run dev即可预览效果。
